A class that extends a base class and implements an interface and also uses a trait to override a method from the base class the class extends.

Base class:

Namespace Lace

class Base {
    
    public function getName()
    {
        return 'danielle';
    }

}

Interface:

namespace Lace

interface Platform {
    
    function getName();
}

Trait:

namespace Lace

trait Smirk {
    
    /*
    public function getName()
    {
        return 'rencia';
    }
    */
}

Class: 

namespace Lace

class Foundation extends Base implements Platform {
    
    // Use the trait

}

Test script:

use Lace\Foundation;

$person = new Foundation();

echo $person->getName();  // should be 'rencia'
